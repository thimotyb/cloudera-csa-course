# Flink Demos

Questa cartella contiene tre percorsi demo distinti.

- `flink/`: lab Flink training vendorizzati (`ride-cleansing`, `rides-and-fares`, `hourly-tips`, `long-ride-alerts`)
- `flink-cli-wordcount/`: demo CLI standalone (download Flink locale, start cluster, run WordCount)
- `flink-cli-windowing/`: demo CLI standalone per tutorial windowing (build Maven Java 21 + start/run/list/cancel)

Per dettagli:

- `flink/README.md`
- `flink-cli-wordcount/README.md`
- `flink-cli-windowing/README.md`
